1. 🚀 Premiers pas avec EduPython

Installation

  1. Va sur edupython.free.fr
  2. Télécharge la version pour Windows
  3. Installe EduPython comme un programme classique

Lancer EduPython

  1. Ouvre EduPython
  2. En haut à gauche, clique sur « Fichier » puis « Nouveau »
  3. Tape ton code à gauche
  4. Clique sur le triangle vert (Exécuter) pour lancer le programme

2. 🧱 Les variables

Une variable, c’est une « boîte » qui contient une valeur.

nom = "Alice"
age = 12

Exemple :

nom = "Léo"
age = 14
print("Nom :", nom)
print("Âge :", age)

🧠 Exercice : Crée 3 variables : prénom, animal préféré, âge. Affiche-les avec print().

3. 🔢 Les types et opérations

Types de données

  • str : Texte
  • int : Nombre entier
  • float : Nombre décimal

Opérations courantes

a = 10
b = 3

print(a + b)   # addition
print(a - b)   # soustraction
print(a * b)   # multiplication
print(a / b)   # division
print(a // b)  # division entière
print(a % b)   # modulo (reste)
print(a ** b)  # puissance

🧠 Exercice : Crée deux variables avec des nombres de ton choix. Affiche leur somme et leur produit.

Commentaires

Utilise # pour un commentaire sur une ligne :

# Ceci est un commentaire

Ou ''' pour des commentaires sur plusieurs lignes :

'''
Ceci est un
commentaire multi-ligne
'''

Comparaisons

a < b     # inférieur
a <= b    # inférieur ou égal
a > b     # supérieur
a >= b    # supérieur ou égal

4. ❓ Les conditions (if)

age = 17
if age >= 18:
    print("Majeur")
else:
    print("Mineur")

Exemple avec plusieurs cas

note = 13
if note >= 16:
    print("Très bien")
elif note >= 10:
    print("Moyen")
else:
    print("Insuffisant")

🧠 Exercice : Crée une variable tirelire. Selon sa valeur, affiche "tu es riche", "ça va" ou "tu es pauvre".

5. 🔁 Les boucles

Boucle for

for i in range(5):
    print("Bonjour", i)

Boucle while

i = 0
while i < 3:
    print(i)
    i += 1

🧠 Exercice :

  • Affiche les nombres de 1 à 10
  • Affiche les nombres pairs entre 1 et 20

La fonction range

  • range(5) → 0 à 4
  • range(2, 6) → 2 à 5
  • range(1, 10, 2) → 1, 3, 5, 7, 9

6. 🧩 Les fonctions

Les fonctions permettent de réutiliser du code.

def bonjour():
    print("Salut !")

bonjour()

Fonction avec paramètre

def dire_bonjour(nom):
    print("Bonjour", nom)

dire_bonjour("Julie")

8. 🎮 Mini-projet : Deviner un nombre

Créer un jeu où l’ordinateur choisit un nombre secret, et l’utilisateur doit le deviner.

import random

secret = random.randint(1, 100)
trouve = False

while not trouve:
    devine = int(input("Devine un nombre : "))
    if devine < secret:
        print("Trop petit")
    elif devine > secret:
        print("Trop grand")
    else:
        print("Bravo !")
        trouve = True
