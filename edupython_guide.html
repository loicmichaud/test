📘 Débuter la programmation en Python avec EduPython

Un guide pas à pas pour les grands débutants

🖥️ Public : Débutants, élèves, autodidactes, enseignants | 📍 Outil : EduPython (Windows) | 📄 Contenu : Bases, exemples, exercices, mini-projet

🧭 Introduction

Bienvenue ! Ce guide s’adresse à celles et ceux qui n’ont jamais programmé.

Tu vas découvrir comment créer de petits programmes avec Python, en utilisant EduPython, un outil gratuit et en français.

Pourquoi Python ?

  • Facile à lire et à écrire
  • Utilisé dans les jeux, sites web, applications, intelligence artificielle…
  • Apprécié dans les écoles, les entreprises et les labos de recherche

Pourquoi EduPython ?

  • Interface simple
  • Conçu pour l’apprentissage
  • Idéal pour Windows

1. 🚀 Premiers pas avec EduPython

Installation

  1. Va sur EduPython
  2. Télécharge la version pour Windows
  3. Installe EduPython comme un programme classique

Lancer EduPython

  • Ouvre EduPython
  • En haut à gauche, clique sur « fichier » puis « Nouveau » pour créer un fichier
  • Tape ton code à gauche, clique sur le triangle vert (Exécuter) pour le lancer
